### 本地预览
```
npm install
安装ruby：https://rubyinstaller.org/downloads/ 
安装ruby的包管理器rubygems：[rubygems](https://rubygems.org/rubygems/rubygems-3.3.25.zip)，下载完成后，解压出来，可以直接双击steup.rb运行，因为前面已经注册了.rb扩展名，会直接调用ruby打开。
cmd下执行：gem install jekyll bundler 安装bundler、jekyll，文档参考：https://www.cnblogs.com/zhaotianff/p/16866153.html
npm start
```
